# توثيق منطق العمل (Business Logic) — نظام لوحات المهام

هذا المجلد يشرح **منطق العمل** (قواعد الأعمال والقرارات المنطقية) الخاص بالمشروع
باللغة العربية. الهدف منه أن يكون المرجع الذي يفهم منه أي شخص — مطوّر أو مدير منتج —
**ماذا يفعل النظام ولماذا اتُّخذت القرارات التقنية بهذا الشكل**، بمعزل عن تفاصيل بناء
المشروع وتشغيله (تلك موجودة في `README.md` و `CLAUDE.md` في جذر المستودع).

النظام عبارة عن لوحة مهام على نمط Trello (Kanban): **لوحات ← قوائم (أعمدة) ← بطاقات**
مع إمكانية السحب والإفلات لإعادة الترتيب.

## كيف تُحدَّث هذه الوثائق؟

هذه الوثائق ليست ثابتة، بل يجب أن تبقى مطابقة للكود. راجع
[`08-maintaining-docs.md`](./08-maintaining-docs.md) للاطّلاع على السياسة المتّبعة:
**أي تغيير في منطق العمل أو أي ميزة جديدة يجب أن يصاحبه تحديث للملف المعني هنا في
نفس الـ Pull Request.** هذه السياسة مُوثّقة أيضًا في `CLAUDE.md` بجذر المستودع كي
يلتزم بها المساعد الآلي تلقائيًا.
